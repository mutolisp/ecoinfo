15  繪圖

R 支援基本的繪圖，包括散布圖(distribution plot)、盒鬚圖(boxplot)、直方圖(histogram)甚至是地圖等。若需要進階的繪圖，可安裝 ggplot2 等第三方套件支援。

15.1 基本繪圖

我們以下圖一來說明 R 繪圖的基本元件及定義 圖一、R 的基本繪圖

A: 繪圖區，呈現點、線等圖之處 B: 圖標題(graph title, R 選項:main) C: X 軸(x-axis) D: Y 軸(y-axis) E: X 變數標題 (x label, R 選項: xlab) F: Y 變數標題 (y label, R 選項: ylab)

15.1.1 盒鬚圖(boxplot)

# 加載 mtcars 資料集，接下來就可以直接
# 使用 mtcars 裡頭的變數
# 未加載 mtcars 要使用 mpg 變數: mtcars$mpg
# 加載之後只需要使用 mpg
attach(mtcars)
boxplot(mpg)
# 繪製完之後卸載 mtcars
detach(mtcars)

15.1.2 直方圖(histogram)

attach(mtcars)
hist(mpg)
detach(mtcars)

圖三、直方圖範例

其他選項： breaks: 直方柱數量(number of bins)，例如 breaks = 10 為繪製 freq: 預設使用出現的頻度，若設定為 FALSE 則會繪製 probability densities

也可以繪製多個不同變數

attach(mtcars)
boxplot(mpg, wt, drat, wt, qsec)
detach(mtcars)

15.3.1 課堂練習 RScript

hdata <- rnorm(100)
# draw histogram using hist()
hist(hdata, axes = FALSE)
axis(1)

box()


### Use examples from Quick-R website
attach(mtcars)
# 單純繪圖，由螢幕輸出圖表
  plot(wt, mpg, type = "p", xlab = "WT (unit)",
       ylab="MPG (miles)")
  # lm 為 linear model (簡單線性回歸)
  abline(lm(mpg~wt), lwd = 2, lty = 2, 
         col = "#ff000099")
  title("Regression of MPG on Weight")
